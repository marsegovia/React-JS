Implementación de una barra de búsqueda
Una barra de búsqueda nos permitirá filtrar una lista de datos en tiempo real basándose en lo que el usuario escribe en un 
campo de texto.
Esto lo logra utilizando un estado de React (useState) para capturar y almacenar cada letra que el usuario tipea en el input.
Antes de mostrar la lista de elementos (productos, artículos, etc.), usa el texto guardado en el estado para crear un nuevo 
array que solo contenga los ítems que coinciden con la búsqueda, haciendo que solo se muestre en pantalla la lista ya filtrada,
que se actualiza automáticamente con cada tecla que el usuario presiona.
Su propósito es mejorar la experiencia del usuario (UX) permitiéndole encontrar rápidamente lo que necesita dentro de grandes 
volúmenes de información. Es una herramienta de interactividad fundamental que hace que la navegación en el sitio sea más
rápida y eficiente, utilizando únicamente la lógica interna de React.

Búsqueda global con Context API
Si queremos que la búsqueda se pueda usar en diferentes partes de la app, podemos almacenarla en Context API.
1️⃣Crear el contexto de búsqueda
📂 context/SearchContext.jsx
import { createContext, useState, useContext } from "react";
const SearchContext = createContext();
export function SearchProvider({ children }) {
const [busqueda, setBusqueda] = useState("");
return (
<SearchContext.Provider value={{ busqueda, setBusqueda }}>
{children}
</SearchContext.Provider>
);
}
export function useSearch() {
return useContext(SearchContext);
}

2️⃣Envolver la app con el SearchProvider
📂 App.jsx
// /src/App.jsx
// Otras importaciones
import { SearchProvider } from "./context/SearchContext";
function App() {
return (
<>
<SearchProvider>
<Routes>
<Route element={<Layout />}>
<Route path="/" element={<h1>Página de Inicio</h1>} />
{/* Otras rutas*/}
</Routes>
</SearchProvider>
</>
);
}
export default App;
