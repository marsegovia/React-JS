Desmenuzando el código
● fetch('/data/productos.json') : Le pide al servidor el archivo productos.json que está en
la carpeta /data/. y nos devuelve una promesa que representa la respuesta del servidor
● then(respuesta => { ... }) : Cuando el servidor contesta, entramos al primer .then. que nos
devuelve un objeto de tipo Response con datos crudos (no JSON todavía).
● console.log('Respuesta cruda del servidor:', respuesta); return respuesta.json(); : Una vez impreso 
por consulta la respuesta, la convierte en un objeto/array de JavaScript.(Esto también devuelve otra promesa).
● then(datos => { ... }) : Ahora sí tenemos los datos ya convertidos en JSON y los mostramos en consola:

● console.log('¡Productos cargados!', datos);
    catch(error => { ... }) : Si algo sale mal caemos en este bloque y al igual que los demás, lo 
    mostramos por consola: console.error('¡Ups! Hubo un error:', error);
● finally(() => {console.log('¡Productos cargados!', datos)} Se ejecuta siempre al final, haya 
funcionado o no.

Integrando todo: useEffect + fetch + useState
Como vimos anteriormente, cuando pedimos datos, estos no aparecen al instante. Puede haber una demora o 
puede que la petición falle. Una buena aplicación debe manejar estas situaciones para que el usuario 
sepa qué está pasando.
Para esto, vamos a usar tres useState:
1. productos: Para guardar los datos cuando lleguen.
2. cargando (loading): Un booleano (true/false) que nos dice si estamos esperando la respuesta
3. error: Para guardar un mensaje de error si algo sale mal.

Ahora juntemos todo en un componente. Dentro de nuestra carpeta componentes, creamos una carpeta y 
su archivo .jsx llamada Productos. En el vamos a escribir el siguiente componente:

    import React, { useState, useEffect } from 'react';
    function Productos({Mensaje}) {
        const [productos, setProductos] = useState([]);
        const [error, setError] = useState(null);
        const [cargando, setCargando] = useState(true);
        useEffect(() => {
            fetch('/data/productos.json').then((respuesta) => {
                if (!respuesta.ok) {
                    throw new Error('No se pudo cargar la información de losproductos');
                }
            return respuesta.json();
            }).then((datos) => {
                setProductos(datos);
            }).catch((error) => {
                setError(error.message);
            }).finally(() => {
                setCargando(false);
            });
        }, []);

        if (cargando) {
            return <p>Cargando productos, por favor espere...</p>;
        }
        if (error) {
            return <p>Error: {error}</p>;
        }
        
        return (
            <div>
                <h1>{Mensaje}</h1>
                <ul>
                    {productos.map((producto) => (
                       <li key={producto.id}>
                            <h2>{producto.nombre}</h2>
                            <img src={producto.imagen} alt={producto.nombre} width="150" />
                            <p>Precio: ${producto.precio}</p>
                        </li>
                    ))}
                </ul>
            </div>
        );
    }
    export default Productos;

Con este if (cargando) y if (error), le damos al usuario una respuesta inmediata sobre el estado de la 
aplicación. ¡Eso es mejorar la experiencia de usuario (UX)!
