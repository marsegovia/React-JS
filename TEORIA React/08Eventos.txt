Eventos: Haciendo que las cosas Reaccionen

Vemos que esteticamente nuestra página va quedando genial, pero ese botón de "Comprar" no hace nada, está de adorno. 
Vamos a comenzar a darle vida. Para eso, necesitamos manejar eventos.

Un evento es cualquier cosa que hace el usuario: un clic, pasar el mouse por encima de algo, escribir en un campo
de texto, etc. React nos facilita muchísimo escuchar y responder a estos eventos.
La sintaxis es muy parecida a HTML pero en camelCase. En lugar de onclick, usamos onClick.

Vamos a modificar nuestro componente Item.jsx para que, cuando hagamos clic en el botón, nos muestre una alerta. 
Para eso creamos una función flecha que nos permita enviar un alert con un mensaje en particular.
Si quiero que mi mensaje, contenga un dato en particular se lo agrego de esta menera: ${dato A Mostrar}. 
    
    // En /componentes/Item/Item.jsx
    export function Item({ nombre, precio, stock }) {
    const CompraClick = () => {// Quiero que se ejecute cuando le doy clic
    alert(`¡Agregaste ${nombre} al chango!`);
    };
    return (
        <div style={{ border: '1px solid #ccc', padding: '15px', borderRadius:'8px' }}>
            <h3>{nombre}</h3>
            <p>Precio: ${precio}</p>
            <p>Stock disponible: {stock}</p>
            {/* Cuando le boton detecta un clic (onClick), se ejecuta la función */}
            <button onClick={CompraClick}>Comprar</button>
        </div>

Ahora nuestro botón es interactivo y pudimos hacer que nuestra aplicación responda al usuario. Otros eventos 
comunes son: onChange, detecta cambios en los inputs, onSubmit, se dispara cuando el formulario es enviado, 
onMouseOver, onMouseOut para manejar los eventos del mouse.

Estados: Dándole memoria a los componentes con usestate
La alerta está buena, pero no es muy útil. Nosotros queremos poder llevar la cuenta de cuántos productos queremos. 
Para eso, nuestros componentes necesitan memoria. Necesitan poder recordar cosas que van cambiando. En React, esa 
memoria se llama estado.
Para manejar el estado en componentes funcionales, React nos da unas herramientas muy buenas llamadas Hooks. 
El primer hook que vamos a ver es useState.

¿Cómo funciona useState?
useState es una función que nos devuelve un array con dos cosas:
1. El valor actual del estado: La "cosa" que queremos recordar (un número, un texto, una lista, un bool, etc.).
2. Una función para actualizar ese valor: La única forma correcta de cambiar el estado.

Vamos a verlo con un ejemplo súper clásico y claro: un contador.

Componente contador

Vamos a nuestra carpeta de componentes y creamos una carpeta llamada Contador. Dentro generamos su componente.

Componente Contador.jsx
// En componentes/Contador/Contador.jsx
// Importamos 'useState' desde React.
import { useState } from 'react';
export function Contador() {
const [contador, setContador] = useState(0);
const incrementar = () => {
setContador(contador + 1); //¡Usamos la función para actualizar el estado!
};
//Continúa!

const decrementar = () => {
setContador(contador - 1);
};
return (
<div style={{ margin: '20px', padding: '20px', border: '1px solid black'
}}>
<h3>Contador de Ejemplo</h3>
<p>Valor actual: {contador}</p>
<button onClick={incrementar}>Sumar +1</button>
<button onClick={decrementar}>Restar -1</button>
</div>
);
}

Explicación paso a paso
Importación de useState
import { useState } from 'react';;
● Como importamos algún componente, desde React importamos el hook useState y cualquier otro.

Declaración del estado
Creamos una variable que desarma el return del useState. Recordemos que dijimos que devolvía un valor y una función.
const [contador, setContador] = useState(0);
● contador: valor actual del estado.
● setContador: función que actualiza ese valor.
● useState(0): inicializa el contador en 0.
Clave: cada vez que llamamos a setContador, React vuelve a renderizar el componente con el nuevo valor.
Por convención, la función del hook se va a llamar set + “el nombre del valor actual del estado”

Funciones de actualización
Creamos dos funciones que van a modificar el valor del estado actual.
const incrementar = () => {setContador(contador + 1);};
const decrementar = () => {setContador(contador - 1);};
● incrementar: suma uno a la variable contador.
● decrementar: resta uno a la variable contador.
● Nunca modificamos la variable contador directamente, siempre usamos setContador.
Renderizado (lo que se ve en pantalla)

return (
<div style={{ margin: '20px', padding: '20px', border: '1px solid black'
}}>
<h3>Contador de Ejemplo</h3>
<p>Valor actual: {contador}</p>
<button onClick={incrementar}>Sumar +1</button>
<button onClick={decrementar}>Restar -1</button>
</div>
);
● Se muestra el valor actual del contador.
● Dos botones permiten al usuario cambiar el valor actual del contador. Con cada clic
disparamos un evento que llama a las funciones generada: incrementar y decrementar

Flujo de funcionamiento
1. El componente se incia → contador comienza en 0.
2. El usuario hace clic en “Sumar +1” → se ejecuta incrementar.
3. setContador actualiza el valor → React vuelve a renderizar.
4. En pantalla aparece el nuevo valor.
5. El mismo ciclo ocurre si se presiona “Restar -1”.

Conceptos clave para llevarse
● useState = Hook para manejar estado en componentes.
● setContador = única forma válida de actualizar el estado.
● Cada cambio en el estado provoca un re-render del componente.
● Estado = memoria interna del componente que cambia con el tiempo.

Estado + evento
Vamos a fusionar los dos conceptos: el evento del clic y el estado de useState, pero dentro de nuestro componente 
Item.jsx. La idea es que cada producto en nuestra vidriera tenga su propio contador independiente.

Uniendo useState y Eventos en Nuestro

Componente Item
Vamos a modificar el archivo Item.jsx para que no solo muestre la información del producto, sino que también nos
permita elegir cuántas unidades queremos agregar a un futuro carrito de compras.

Así es como quedaría el componente Item.jsx

// En /componentes/Item/Item.jsx
import { useState } from 'react';
export function Item({ id, nombre, precio, stock }) {
    const [cantidad, setCantidad] = useState(0);
    const incrementar = () => {
        if (cantidad < stock) {
            setCantidad(cantidad + 1);
        }
    };

    const decrementar = () => {
        if (cantidad > 1) {
            setCantidad(cantidad - 1);
        }
    };

const agregarAlCarrito = () => {
alert(`Agregaste ${cantidad} unidades de ${nombre} al carrito.`);
}

return (
<div style={{ border: '1px solid #ccc', padding: '15px', borderRadius:
'8px', textAlign: 'center' }}>
<h3>{nombre}</h3>
<p>Precio: ${precio}</p>
<p>Stock disponible: {stock}</p>

<div style={{ display: 'flex', alignItems: 'center', justifyContent:
'center', margin: '10px 0' }}>
<button onClick={decrementar}>-</button>
<p style={{ margin: '0 10px' }}>{cantidad}</p>
<button onClick={incrementar}>+</button>
</div>
<button onClick={agregarAlCarrito}>Agregar al Carrito</button>
</div>
);
}
