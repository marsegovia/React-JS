💿 WraithMoon Records
Una tienda online especializada en la venta de música (vinilos/CDs), desarrollada con React.js.

🚀 Tecnologías utilizadas
Framework: React.js (Vite)

Gestión de Estado: React Context API

Estilos: CSS Modules

Iconos: FontAwesome

Navegación: React Router DOM

✨ Características principales

Carrito de Compras: Sistema de gestión de estado global para agregar, eliminar y calcular subtotales.

Persistencia: La información se mantiene durante la navegación mediante Context API.

🛠️ Instalación y ejecución
Clona el repositorio: git clone https://github.com/marsegovia/React-JS

Instala las dependencias: npm install

Ejecuta el servidor de desarrollo: npm run dev
